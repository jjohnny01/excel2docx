# exceltodocx
using openpyxl to extract values from a spreadsheet 


**Installation:**
run the install.sh script
./install.sh

**Usage:**
'excel2docx' + 'name of excel file' + 'Sheet1(or the name of the spreadsheet)(it's case sensitive)' + 'name of output file'
